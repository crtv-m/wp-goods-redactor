# Goods Redactor Plugin

Goods Redactor is a WordPress plugin designed to manage products within WooCommerce stores. The plugin offers functionalities for bulk editing product categories and setting sale prices.

## Installation
1. Download the plugin archive or clone the repository into `wp-content/plugins/` directory of your WordPress site.
2. Activate the plugin through the 'Plugins' menu in WordPress.

## Requirements
- WordPress version 5.6 or higher.
- PHP version 7.2 or higher.
- WooCommerce must be installed and activated for the plugin to function correctly.

## Features
- **Bulk Category Editing**: Select multiple products and add them to a new or existing category.
- **Setting Sale Prices**: Specify a new sale price and apply it to all selected products.

## Usage
1. **Setting Filters**:
    - Navigate to the "Goods Redactor" page in your WordPress admin panel.
    - Choose product categories for filtering.
    - Enter keywords to search for specific products.

2. **Selecting Products**:
    - Use the table to select specific products you want to edit.

3. **Bulk Editing**:
    - Set a new category for selected products or specify a new sale price.
    - Click "Save Changes" to apply all edits.

## Notes
- The plugin requires an active WooCommerce installation for access to basic settings and product handling functions.
- It's recommended to regularly back up your site before making extensive changes using this plugin.

## Frequently Asked Questions
1. **How do I install the plugin?**
   - Follow the installation steps mentioned above.

2. **Does the plugin work without WooCommerce?**
   - No, WooCommerce must be installed and activated for the plugin to function correctly.

# Плагин Goods Redactor

Goods Redactor — это плагин для WordPress, предназначенный для управления товарами в магазинах WooCommerce. Плагин предоставляет функционал для массового редактирования категорий товаров и установки цен со скидкой.

## Установка
1. Скачайте архив плагина или клонируйте репозиторий в директорию `wp-content/plugins/` вашего сайта WordPress.
2. Активируйте плагин через меню 'Плагины' в WordPress.

## Требования
- WordPress версии 5.6 или выше.
- PHP версии 7.2 или выше.
- WooCommerce должен быть установлен и активирован для корректной работы плагина.

## Функции
- **Массовое редактирование категорий**: Выберите несколько товаров и добавьте их в новую или существующую категорию.
- **Установка цен со скидкой**: Укажите новую цену со скидкой и примените её ко всем выбранным товарам.

## Использование
1. **Настройка фильтров**:
    - Перейдите на страницу "Goods Redactor" в админ-панели WordPress.
    - Выберите категории товаров для фильтрации.
    - Введите ключевые слова для поиска конкретных товаров.

2. **Выбор товаров**:
    - Используйте таблицу для выбора конкретных товаров, которые вы хотите редактировать.

3. **Массовое редактирование**:
    - Установите новую категорию для выбранных товаров или укажите новую цену со скидкой.
    - Нажмите "Сохранить изменения", чтобы применить все изменения.

## Примечания
- Для работы плагина требуется активная установка WooCommerce, чтобы иметь доступ к базовым настройкам и функциям обработки товаров.
- Рекомендуется регулярно делать резервные копии вашего сайта перед выполнением обширных изменений с использованием этого плагина.
